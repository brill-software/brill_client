Brill Framework

The Brill Framework is Open Source and you are free to use it however you like. Its distributed under the MIT license. You can run it on development workstations, your own in-premise servers and own cloud servers, for free. You can run servers handling millions of users, for free. You have the full source code and are not dependent on Brill Software. You can modify the source code and re-compile.

Brill CMS

The Brill CMS runs as a Web Application on top of the Brill Framework. The application is distributed under the Brill Software Apps License and is a charged for product.

During the Beta Phase, you are welcome to use the Brill CMS for free. The Beta Phase is expected to last until the end of 2023.

After the Beta Phase, the proposed charge is €120 per CMS user per month. The Brill CMS will be free for Open Source projects and free for personal use. It will also be free for anyone that is unable to pay or can't afford the license charge.
