Brill Storybook demostrates React Components for Layout, Drag and Drop ( DND ), JSON editor, input fields and data tables

Storybook

The Storybook provides a means of displaying and testing existing and new components.

The left panel displays a list of pages using the TreeView component. The top right panel displays the currently selected component using the EmbedPage component. In the bottom right panel the component JSON is displayed. The panels are created using the ResizablePanels component.

The component you are currently viewing is a ConfluencePage component. The text is held in a XML file that that uses the Confluence Internal Storage Format.
